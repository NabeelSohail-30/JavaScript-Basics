Timer Methods

  • setInterval(functionName, Milliseconds)

    • Executes the given function in every given Milliseconds - Returns a number as unique Id

  • clearInterval(setIntervalMethodId)

    • to stop setInterval method executions

  • setTimeout(functionName, Milliseconds)

    • Executes the given function after the given Milliseconds only once - Returns a number as unique Id

  • clearTimeout(setTimeoutMethodId)

    • to cancel setTimeout method execution
